# 🛒 SMART MALL AI

### 🔍 Content-Based Image Retrieval (CBIR) Product Recognition System

🚀 **Smart Mall AI** is an AI-powered **Content-Based Image Retrieval (CBIR)** system designed for smart retail environments.
The system identifies products from images by extracting visual features and comparing them with a database of product images.

This project demonstrates how **Computer Vision, Machine Learning, and Image Retrieval techniques** can be applied to improve **product discovery, automated recognition, and smart retail experiences**.

---

# ✨ Key Features

📸 **Image-Based Product Search**
Users can upload or capture an image of a product.

🧠 **CBIR Algorithm Implementation**
Retrieves similar products based on visual features instead of text metadata.

🔍 **Feature Extraction & Similarity Matching**
Extracts image features and compares them with stored dataset features.

⚡ **Real-Time Product Identification**
Efficient retrieval system for quick product matching.

🌐 **Web Interface**
Interactive interface built using **Flask** for user interaction.
